All motion in Magic Learning Adventure (floating clouds, sparkles, fireflies,
rainbow shimmer, confetti, coin bursts, page transitions, button ripples,
breathing circle, etc.) is implemented with CSS keyframes in /style.css and a
small canvas-confetti helper in /script.js.

This keeps the app dependency-free and fast — no external animation library
or Lottie files are required. If you want to add Lottie/JSON animations
later, this is the folder to keep them in.
